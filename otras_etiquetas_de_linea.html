Otras Etiquetas de Linea

span -> es un contenedor de linea, equivalente a div, se suele usar para encerrar palabras o pequeos textos y darles estilo a traves de CSS o localizarlos desde JavaScript. Semanticamente no significa navegador

q -> es el equivalente a blockquote, significa quote, por eso el de bloque se llama block - quote. Sirve para poner citas pero en Linea

code -> sirve oara encerrar codigo que queremos representar visualmente, suele ir unido con la etiqueta pre
